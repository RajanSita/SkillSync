> Transform your learning journey with AI-generated, personalized roadmaps

## Overview

**SkillSync** is a professional-grade learning platform that revolutionizes skill acquisition by generating **personalized learning roadmaps** using advanced AI algorithms. Built to address the critical challenge of structured learning in today's fast-paced digital world, it provides users with a clear, actionable path from beginner to expert.

## 🛠️ Tech Stack

### **Frontend**
- **Core**: React 19.0
- **Styling**: Tailwind CSS (PostCSS/Autoprefixer)
- **Icons**: Lucide React
- **Icons & Graphics**: Custom SVG & High-res Mock assets
- **State Management**: React Context API & Modern Hooks

### **Backend & Services**
- **Database**: Airtable API for roadmap persistence
- **AI Integration**: Relay Planner API for intelligent roadmap generation
- **Auth Simulation**: Integrated Mock Google Authentication service
- **Storage**: LocalStorage & SessionStorage for immediate state persistence

## 🚀 Getting Started

### **Prerequisites**
- Node.js (v18+)
- npm or yarn

### **Installation**
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables in `.env` (Airtable API Key, Base ID)
4. Start the development server: `npm start`
